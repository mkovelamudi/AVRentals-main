# 2022 Spring CMPE 281 Project - AVRentals

This is the project folder for AVRental cloud for San Jose State University, Spring 2022 semester CMPE 281.

# Persona

System will be having  three different users  the car owner and business user and  the system admin

* Business User
    * Books Autonomous Vehicles as a taxi rides.
* Car Owner
    * Can register their autonomous vehicles into the system.
* System Admin
    * Have an overlook over the system.
